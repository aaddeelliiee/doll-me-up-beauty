DOLL ME UP BEAUTY — PLACEHOLDER WEBSITE
========================================

WHAT IS INCLUDED
- index.html: the page content and search/social metadata
- styles.css: all colors, layout, typography, and responsive styling
- script.js: the current year and subtle reveal animation
- netlify.toml: redirects the .beauty and www domains to dollmeupbeauty.com
- robots.txt and sitemap.xml: basic search-engine files

QUICK EDITS
1. Open index.html in any text editor.
2. Search for "EDIT HERE". There are clearly marked areas for:
   - Page title and description
   - Service teaser text
   - Email, Instagram, and location
3. The current placeholders are:
   - hello@dollmeupbeauty.com
   - @dollmeupbeauty
   - Your City, State
4. Save the file and open index.html in a web browser to preview it.

NETLIFY DEPLOYMENT
1. Sign in to Netlify and choose "Add new project" / "Deploy manually".
2. Drag this entire folder—or the supplied ZIP—into the upload area.
3. In Domain management, add dollmeupbeauty.com as the primary domain.
4. Add dollmeup.beauty, www.dollmeupbeauty.com, and www.dollmeup.beauty as domain aliases.
5. Follow Netlify's DNS instructions while keeping all Neo email MX/TXT records intact.

NOTES
- This is a completely static site. It does not use Supabase or a database.
- The fonts load from Google Fonts. If they cannot load, the site uses standard fallbacks.
- The contact button opens the visitor's email app; there is no stored contact form.
